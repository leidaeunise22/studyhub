Step 1: Install Streamlit and Required Libraries

First, ensure you have Python installed. Then, install Streamlit and other necessary libraries using pip:

pip install streamlit google-generativeai requests PyPDF2

Create a CSS file named style.css for your app’s styling. Here’s a basic example:

Step 5: Run Your Streamlit App

Open your terminal or command prompt and navigate to the directory where your app.py is located. Then run:

streamlit run app.py

Your web app should open in a new browser tab!
